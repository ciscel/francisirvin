How to view
Open index.html in any modern browser. All pages, styles, and scripts are included locally.

How to publish
Upload the full francis_portfolio folder to Netlify, GitHub Pages, Cloudflare Pages, cPanel, or another static web host. Set index.html as the home page.

Resume
The original Word resume is included at assets/Francis_Amojelar_Resume.docx. The Resume page also has a print-friendly layout that can be saved as PDF from the browser.

Contact form
Because this is a static website, the form opens the visitor's default email app and addresses the message to francisamojelar@icloud.com. For direct submissions, connect the form to Formspree, Netlify Forms, or a custom backend.
